# Universo Engine

C++ 3D Engine with no level editor (code only). Currently in (early) development.

## How to download and build (Windows)

```
git clone --recurse-submodules https://github.com/MatheusZickuhr/UniversoEngine.git
cd UniversoEngine
premake5 vs2019
```
